# Microblog

v 1.0.0 介绍

这是用 node + mongodb + bootstrap编写的一个小型微博，
只实现了注册和登录功能，并且根据这两种状态设置了一些权限，
微博功能就这些，只是当作自己对这一点东西的巩固复习而已。

查看注意、、

依赖node，自己的环境是 node 
依赖mongodb储存用户数据，需要自己创建个db文件夹存储数据然后连接数据库才能打开网站！！！

v 1.0.1
